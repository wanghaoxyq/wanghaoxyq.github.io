Timer使用经验

在之前的工作中，因为需要爬虫能够定时执行，所以使用了Java本身提供的Timer类。为了避免遗忘，记录于此。

Timer是Java本身所包含的工具类，其引用路径为java.util.Timer；在使用中，我们还需要引用一个类，java.util.TimerTask，用来为Timer指定工作任务。

要定时执行某项任务其实非常简单，只需要创建一个TimerTask对象，设置好任务后，将其传递给Timer的schedule方法即可。
schedule可以接收不同的参数，表示不同的定时方式：

  • Timer.schedule(TimerTask task, Date time) 安排任务在指定的时间执行。
  • Timer.schedule(TimerTask task, Date firstTime, Long period) 安排任务在指定的时间开始执行，并以指定时间为周期重复执行。
  • Timer.schedule(TimerTask task, long delay) 安排任务在指定的延迟后执行。
  • Timer.schedule(TimerTask task, long delay, long period) 安排任务在指定的延迟后开始执行，并以指定时间为周期重复执行。
