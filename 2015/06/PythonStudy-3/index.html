  • Python学习笔记
  • fishc.com

1. Python常用操作符

  • a = a + 3可以简写为a += 3,b = b - 1可以简写为b -= 1

  • python支持连续赋值，a = b = c = d = 10给a、b、c、d四个变量均赋值为10

  • 如果需要使用python2中的整型除法，那么使用//作为除号（地板除法）

  • python中的幂运算：使用运算符**

  • 优先级：先乘除，再加减
    幂运算比其左侧的一元运算符优先级高，比其右侧的一元运算符优先级低

  • Python支持3 < 4 < 5这样的语法，其实质是3 < 4 and 4 < 5

  • 优先级列表：
  1. 幂运算：**
  2. 正负号：+x -x
  3. 算术操作符：* / // + -
  4. 比较操作符：< <= > >= == !=
  5. 逻辑运算符：not and or

同一级的，乘除比加减高，not比and高，and比or高

  • 内置函数range()
    语法：range([start,] stop[, step=1])
    –这个BIF有三个参数，其中用中括号括起来的两个表示这两个参数是可选的
    –step = 1表示第三个参数的默认值是1
    –range这个BIF函数的作用是生成一个从start参数的值开始到stop参数的值结束的数字序列
    –step是生成的数字序列的等差间隔（步距）

可以发现range()返回的数字序列默认从0开始，并且不包含stop的值。
list()接受参数并返回一个元组
