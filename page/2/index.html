Python学习（38）：爬虫：隐藏

1. Python模拟浏览器访问

  • class urllib.request.Request(url, data=None, headers={}, origin_req_host=None, unverifiable=False, method=None)
    查询文档，可以看到该方法的用法。
    我们这里关注的主要是参数headers，它是一个字典，它的作用与里面的每一个键值对都被add_header()调用一致。该参数通常用于“伪装”User-Agent。例如：Mozilla FireFox的User-Agent的格式为：’Mozilla/5.0 (X11; U; Linux i686) Gecko/20071127 Firefox/2.0.0.11’，而用Python的urllib库访问的User-Agent格式为：’Python-urllib/2.6’（on Python 2.6）

  • 可以通过两种途径设置headers参数：

  1. 直接给headers参数赋值，传递给Request
  2. 在Request生成之后调用add_header()方法将其加进去

Python学习（35）：模块学习：像极客一样思考

  • Python的设计哲学：优雅、明确和简单
  • Python开发者哲学：用一种方法，最好是只有一种方法来做一件事
  • Python的实际开发中，最好使用现有的模块，可以节约大量的精力、时间
  • Python自己带着电池（Batteries included）：Python标准库。Python标准库中包含一般任务所需要的模块，我们需要的是知道如何探索它
  • Python不仅带着电池，还带着充电器：Python文档
  • Python增强建议书（PEP）：http://www.python.org/dev/peps/
  • 如何快速学习一个模块？
  1. 在Python命令行中导入该模块，例如：import timeit
  2. 查看该模块的文档：timeit.__doc，可以看到该模块的简介。但是直接这么看是没有格式的，最好使用print(timeit.\doc__)查看
  3. dir(timeit)可以看到模块内置的变量、函数、类
  4. timeit.__all 可以帮助我们过滤掉我们不需要关心的属性，返回一个我们需要关心的属性（函数、类、变量）的列表。但是不是所有的模块都有\all__属性，需要先用dir方法查看。
  5. 使用 from 模块名 import * 的方式导入模块时，只会导入__all属性的列表中有的属性；如果没有\all__属性，则会导入所有属性
  6. timeit.__file__属性指明了模块的源代码所在的位置
  7. help(timeit)方法查看到的文档比__doc__属性要详细，比官方文档要简单
  8. 看官方文档
  • 快速成长：
  1. 多写代码
  2. 多看高手写的代码

Python学习（36）：爬虫：基础

1. 网络爬虫

  • 网络爬虫从互联网上抓取数据

  • Python如何访问互联网？使用模块：urllib
    urllib实际上是一个包，它包含了几个模块：
    urllib.request，urllib.response，urllib.error，urllib.parse，urllib.robotparser

  • URL的一般格式为（带方括号[]的为可选项）：
    protocal://hostname[:port]/path/[;parameters][?query]#fragment

  • URL由三部分组成：
    – 第一部分是协议：http，https，ftp，file，ed2k，….等等
    – 第二部分是存放资源的服务器的域名系统或IP地址（有时候要包含端口号，各种传输协议都有默认的端口号，如http的默认端口为80）
    – 第三部分是资源的具体地址，如目录或文件名等
